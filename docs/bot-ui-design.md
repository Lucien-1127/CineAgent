# ConeAgent Bot 設計文件

## 1️⃣ 介面設計

### 歡迎訊息模板

當使用者按 `/start` 或第一次對話：

```
🎬 歡迎來到 ConeAgent — 你的 AI 創意副導！

給我一個想法，我幫你完成：
  腳本設計 → 圖片生成 → 影片製作

👉 直接跟我說你想做什麼影片，
   或點下面的選項開始：

[ 🎥 開新專案 ] [ 🎨 風格參考 ]
[ ❓ 怎麼用 ]   [ 📚 歷史作品 ]
```

### 引導對話流程

```
User: 「我想做一支影片」
Bot: 「沒問題！先幫你聚焦幾個方向：

  這支影片主要想傳達什麼感覺？
  🎭 A) 故事/劇情
  🌆 B) 城市/風景
  🎮 C) 科幻/奇幻
  📦 D) 產品/展示
  ✏️ E) 自己說

  點選或打字告訴我 👇」

User: 「B) 城市風景」
Bot: 「好選擇！城市風景的話：

  想要哪種氛圍？
  🌃 A) 繁華夜景
  🌅 B) 黃昏日落
  🌧️ C) 雨天憂鬱
  ☀️ D) 晴天活力

  長度呢？
  ⏱️ 15秒 / 30秒 / 60秒」
```

## 2️⃣ 選單設計

### 指令列表

| 指令 | 功能 |
|------|------|
| `/start` | 顯示歡迎畫面 + 主選單 |
| `/new` | 開啟新影片專案 |
| `/style` | 切換視覺風格主題 |
| `/status` | 查看當前專案進度 |
| `/history` | 瀏覽歷史作品 |
| `/share` | 發布到 Telegram / X |
| `/settings` | 個人化設定 |
| `/help` | 完整使用說明 |

## 3️⃣ 串接功能

### 已串接 API

| API | 用途 | 狀態 |
|-----|------|------|
| Agnes 2.0 Flash | 主要推理 | ✅ |
| DeepSeek V4 | 複雜推理降級 | ✅ |

```
使用者想法
    │
    ▼
┌──────────────┐
│ 意圖分類      │ ← Agnes 2.0 Flash
│ (腳本/風格/   │
│  查詢/設定)   │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 任務拆解      │ ← 子代理分配
│ Script Agent │
│ Image Agent  │
│ Video Agent  │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 執行生成      │
│ Agnes / DS   │
│ 圖片 + 影片   │
└──────┬───────┘
       │
       ▼
┌──────────────┐
│ 產出交付      │
│ Telegram 推送 │
│ X 發布 (選)   │
│ 本地儲存      │
└──────────────┘
```

### Telegram Inline Keyboard 格式

按鈕選單透過 Telegram API 的 `reply_markup` 傳送：

```json
{
  "reply_markup": {
    "inline_keyboard": [
      [{"text": "🎥 開新專案", "callback_data": "new_project"}],
      [{"text": "📊 當前進度", "callback_data": "status"}],
      [{"text": "🎨 風格切換", "callback_data": "style"}],
      [{"text": "❓ 幫助", "callback_data": "help"}]
    ]
  }
}
```

按鈕回調 (callback_data) 清單：
- `new_project` — 開始新專案
- `status` — 查看進度
- `style` — 切換風格
- `style:cyber` — 賽博龐克
- `style:japan` — 日系唯美
- `style:vintage` — 復古電影
- `style:fantasy` — 史詩奇幻
- `share:telegram` — 發布到 TG
- `share:twitter` — 發布到 X
- `settings` — 開啟設定
- `help` — 使用說明
