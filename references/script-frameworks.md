# 📚 腳本創作框架參考

## 常見腳本結構

### 1. 三幕劇結構 (Three-Act Structure)

```
第一幕: Setup (25%)
  ├─ 開場 / 日常世界
  ├─ 催化事件 (Inciting Incident)
  └─ 關鍵抉擇
        ↓
第二幕: Confrontation (50%)
  ├─ 上升行動
  ├─ 中點轉折 (Midpoint)
  └─ 谷底 (All Is Lost)
        ↓
第三幕: Resolution (25%)
  ├─ 最後衝刺
  ├─ 高潮
  └─ 結局
```

**適用：** 3 分鐘以上故事性影片

### 2. 救貓咪節奏表 (Save the Cat! Beat Sheet)

Blake Snyder 的 15 拍節奏表：

```
1. 開場 (1%)        — 主角日常
2. 主題陳述 (5%)     — 隱含主題
3. 設定 (1-10%)     — 關鍵關係
4. 催化劑 (10%)     — 改變一切的事件
5. 爭論 (10-20%)    — 主角猶豫
6. 第二幕銜接 (20%)  — 進入新世界
7. B故事 (22%)      — 副線/關係
8. 趣味遊戲 (20-55%) — 探索新世界
9. 中點 (55%)       — 重大勝利/失敗
10. 壞蛋逼近 (55-75%) — 壓力增加
11. 一無所有 (75%)   — 谷底
12. 靈魂暗夜 (75-85%) — 絕望時刻
13. 第三幕銜接 (85%) — 頓悟
14. 結局 (85-99%)   — 高潮
15. 最終畫面 (99-100%) — 新常態
```

### 3. 短影片鉤子結構

```
0-3s    Hook: 反轉預期 / 問題 / 震撼畫面
3-10s   Retention: 展示核心內容
10-20s  Value: 提供資訊/娛樂價值
20-30s  CTA: 引導互動（留言/分享/追蹤）
```

## 短影片腳本類型

### 產品展示型
```
Hook (3s) → 痛點 → 產品出現 → 解決方案 → CTA
```

### 教學/知識型
```
Hook (3s) → 問題 → 3 個重點 → 總結 → CTA
```

### 故事型
```
Hook → 日常 → 衝突 → 解決 → 感悟
```

### ASMR/沉浸型
```
無Hook → 環境音建立 → 視覺流 → 淡出
```

## 跨鏡頭連續性要點

1. **視線匹配** — 角色視線方向在連續鏡頭中一致
2. **動作匹配** — 動作在剪接點前後連續
3. **位置匹配** — 場景中物體相對位置一致
4. **音調匹配** — 情緒/色調/光線延續
5. **180 度法則** — 不越軸拍攝
